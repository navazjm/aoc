# AoC 2015 Solutions

## Directory Tree

```
project-root/2015/
|-- src/
| |-- day_XX/
| | |-- main.cpp
| | |-- input.txt
| | |-- sample.txt
|-- README.md

```

## Run from source

- CD to desired day

```sh
cd path/to/aoc/2015/src/<day_XX>
```

- Update input.txt with your unique input provided by AoC

- Compile day_XX/main.cpp

```
g++ --std=c++17 main.cpp -o out
```

- Run the program. Pass in input txt as command line arg

```sh
./out input.txt
```
